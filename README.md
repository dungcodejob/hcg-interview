## Development Setup

### Prerequisites

- Install [Node.js] which includes [Node Package Manager][npm]
- Angular Cli v16
- Nestjs/cli v10

### Setting Up a Project
Install dependencies for server:

```
cd server
npm install
```

Start nestjs server:

```
npm run start 
```

Install dependencies for client:
```
cd client
npm install
```

Start client server:

```
ng serve
```
